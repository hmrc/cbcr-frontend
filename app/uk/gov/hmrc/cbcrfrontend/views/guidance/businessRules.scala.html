Register and report using the country-by-country reporting service.

If you’re a relevant multinational group, you must send HM Revenue and Customs (HMRC) a country-by-country report, detailing your business activities.

You’ll use the XML (extensible markup language) schema provided by the Organisation for Economic Co-operation and Development (OECD) for this report.

However, HMRC has its own procedures, detailed below, which you must follow.

OECD schema

You must generate a valid XML that conforms to the OECD schema and send it to HMRC using the reporting service.

Further information

Before completing the schema, you must:

  • be registered to use the online service
  • have read the reporting guidance

If you want to send a report when you’re not required to, check whether you can do so.

You can find more details on the above in the International Exchange of Information Manual.
